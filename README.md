## 📥 Installation

Clone the repository

```bash
git clone https://github.com/Victor-Chilenje/victor-portfolio.git
```

Navigate into the project

```bash
cd victor-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
